# HA_Felshare_Diffuser
This component is a *custom integration* for Home Assistant. It is not an official Felshare product.  It enables control of Felshare "smart" diffusers (and compatible devices) via the *Felshare CloudAuthentication via Felshare API to obtain tokens. Connection to *MQTT over WebSockets (TLS)* in the cloud.Publishing and receiving states and commands
